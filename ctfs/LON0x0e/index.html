Anonymity
This CTF is about recovering personally-identifying information from datasets.

Each challenge and its instructions is contained in a separate zipfile. The solution to each challenge is a piece of information which is the password to the next challenge.

There are a few different ways to work with the csv-formatted data, depending on what you are most familiar with. Importing into SQLite, command line tools, or a spreadsheet program are all valid approaches.

Brute-forcing is explicitly allowed in one of the later challenges, but until then, it is not. Measures have been put in to ensure it will not work.

The challenges are based around a fictional clinic in the USA. All data herein is fictitious.
